# JobForProgrammistC-
4/4 заданий сделано: обработка изображения, медиа-плеер, резервное копирование и сжатие-разжатие файлов
1)Image editor - при запуске сборки открывается windows form с кнопками добавления картинки и накладывания эффектов
2)MediaPlayerV2.0 - медиа-плеер сделанный на WPF
3)ConsoleBackup - создан класс SettingCopy, в котором в source[] сохраняются исходные папкм, в target хранится целевая папка. Данный класс сохраняется в json и также достайтся из него. программа копирует из исходных папок файлы и сохраняет в целевую.
4)GzipCompress - открывается в командной строке при наборе параметров: GzipCompress compress source target или GzipCompress decompress source target. где source - имя исходного файла, target - имя архива
